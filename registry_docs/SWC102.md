#### 过时的编译器版本
[CWE-937](http://cwe.mitre.org/data/definitions/937.html):使用存在已知漏洞的组件

#### 描述
使用过时的编译器版本可能会出现问题，特别是如果存在影响当前编译器版本的公开披露的错误和问题。

#### 补救方法
建议使用最新版本的 Solidity 编译器。

    uint public x = 1;
}
